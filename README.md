# STS2 Relics

A Slay the Spire mod that adds new relics to the game.

## Features

### Campfire Cooking
- **Uncommon Tier Relic**
- Allows you to cook at campfires
- Removes 2 random cards from your deck
- Grants 9 Max HP as a reward

## Installation

1. Compile the mod using Maven: `mvn clean package`
2. Copy the resulting JAR file to your Slay the Spire mods folder
3. Launch the game with BaseMod and ModTheSpire installed

## Development

This mod requires:
- Java 8
- Maven
- Slay the Spire (desktop-1.0.jar)
- BaseMod (5.49.0)
- ModTheSpire
